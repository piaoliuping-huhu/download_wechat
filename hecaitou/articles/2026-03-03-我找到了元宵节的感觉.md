# 我找到了元宵节的感觉

- **日期**: 2026-03-03
- **链接**: https://www.hecaitou.com/2026/03/Tuning-Into-the-Lantern-Festival.html

---

从我知道元宵节一直到昨天，我对元宵节都完全无感。无论是上学还是上班，春假只有七天。等你带着大年初七的脸返回课桌，返回工位，又连续干了七天，然后元宵节来了。怎么可能有节日的感觉？古时候这是个青年男女相会的节日，「淫奔不禁」，而如今是个元宵促销日，这两者怎么相比？
对了，这里我还要多说一句。以前有个人人都玩的小游戏叫做「偷菜」，前段时间又再次复活。偷菜其实也是源自元宵节的风俗，具体说是辽金时期，那时候元宵节要「放偷」，也就是从正月十三开始，连续三天允许群众相互偷盗，「放国人做贼三日」，从蔬菜到人，都在盗窃之列，盗窃的财物价值在十贯以下不予追究，后续双方也可以商量如何赎回。
现在的元宵节距离春假太遥远，很多时候如果不是嘴里被强塞了元宵，或者看到网上元宵党大战汤圆党，我都意识不到有这个节日。
今年的情况有点不一样，今年春节我决定开始听 CD。
我先去网上买了一台新品国货便携 CD 机，北京本地发货，春节不打烊，还可以提供闪送服务。几个小时后 CD机到了，这时候我才发现先前我忽略了一个很重要的问题：我还没有 CD 碟片。又跑到网上，找了一堆碟店下单，无一例外告诉我：对不起，初七复工之后发货。
没办法，我只能跑去闲鱼，找本地卖家淘二手 CD 碟。北京这一点很好，很多人并不回家过年，或者就是北京人。于是，我找到了卖家，别人直接给我闪送碟片过来，这样我起码有了三五张可以听的音乐碟片。
在等闪送的过程里，我百无聊赖开始翻闲鱼......和你猜想的一样，我看中了一台老式 Sony CD 随身听，到今天起码也有 20 岁了。卖家在广东的一个二级市还是三级市，春节期间也营业，人很好，直接发了顺丰。事情从这个时刻开始，就起了变化：
还记得吗？之前我都是用同城闪送，任何东西就是几小时到家，根本不受春节影响。而我买的那台老式 CD 随身听在外地，哪怕是用了顺丰，一站站那么送过来，也花了三天半时间。
春节都快结束的时候，机器终于到家。我兴高采烈地接好线，开始测试机器。还没听完一首歌，CD 随身听的液晶屏幕就不再显示了。各种方法都用尽，推测应该是内部排线松脱。欢喜没有 5 分钟，我又请回刚刚送货上门的顺丰小哥，把机器重新快递回去。
卖家那边听说了消息，回复我说：没事，我再给你寄一台新的去。一天过去了，没有任何动静，没有任何一句话。
第二天，我问他怎么说。他说他在最后检查一下新机器，确定没有问题了，晚些时候就给我寄出去。又一天过去了，没有任何动静，没有任何一句话。
第三天，我问他有没有邮寄。他说，他马上会接到一个日本包裹，里面是海淘来的一堆新机器。其中一台和我要的机器是同型号，非常新，他准备邮寄这一台给我。我问他具体什么时候，他说快了，就这一两天。
第四天，我看我的顺丰快递已经签收，就问他收到包裹没有。一天过去了，没有任何动静，没有任何一句话。
我耐着性子，又等了一天。依然没有任何动静，没有任何一句话。
第六天，我其实有点慌了，觉得事情不大对头。就问他：今天能邮寄出来吗？他回了一段视频，向我展示一台品相很新的机器，然后展示了我邮寄的包裹，告诉我说新机器明天就发出。
第七天，约定的「明天」到来，什么都没有发生，没有任何动静，也没有任何一句话。
第八天，我再次问：今天能邮寄出来吗？他回答说，对不起，昨天有事，忙不过来，今天马上就发。我已经完全不抱任何希望，准备放弃交易，反正这时候北京的CD 随身听卖家早已经回来，几个小时之内我就可以得到任意我想要的型号。消息突然到来：已发货，订单号后续发你。
我查了一下顺丰小程序，这一次的确是发货了，而且快递小哥已经收取包裹开始运送。
这时候我坐下来，冷静地回想了全过程，思考究竟发生了什么，为什么在 21 世纪的第 26 年，还会发生这种耗时两周的网络交易行为？然后，我就是想买一台古董 CD 随身听，给我安排这样一出，上面究竟是一种什么考虑？是想通过这些周折具体告诉我一点什么信息？
然后我突然顿悟：我在过七天的春节，但是广东那个哥们在自己家乡，他是在过十五天的春节。我从大年初八开始，就期待一切交易、一切服务都恢复正轨，在北京的确是这样的，全国很多地方也同样如此，我网购的 CD 碟片都是在大年初八当天从合肥、南京、上海、深圳发出。但不是所有人，所有地方都如此。
广东哥们依然在过春节。所以，他每天对同一个顾客做一件事，回复一次，这已经算是在加班了。他回完我之后，就不再看，我后续发什么都是未读状态。第二天，我再发一条消息过去，他才会点开看，再回一句，然后又把手机扔到一边继续过节。
想明白这一点，我突然对元宵节有了点感觉。认定春节是从除夕一直持续到正月十五的人，元宵节就是个真正的节日。在之前的十五天里都是过年，一切都不需要着急，一切都可以等等再说，一切和过年相比也没有那么重要。这样的一颗心到了元宵节才需要专门找个日子，庆祝一番，然后把它收回来，接下来恢复以往的节奏。
我们生活在同一个国家里，但是我们生活在不同的时间节奏里，生活在不同的风俗传统里。相对应的，是我们的心彼此不同。广东哥们拥有我所没有的闲适舒缓心境，而我呢，看到包裹已经发出，第一时间就在订单页面里找入口，看有没有一个交钱提速选项，把普快提升到特快。没有看到选项，就想给顺丰反馈意见，要求新增功能---我就是这种心态。
今天是元宵节，包裹正在离开湖北黄冈继续北上，距离我还有 1200 公里。我已经不那么热切地期待，我已经觉察到了其中的荒谬。相反的，我通过那个广东哥们这几天来的反应，找到了一点元宵节的感觉，找到了一点对时间的正常感觉。
我也打算把所有事情先放一放，先去过个节，比如说去朋友家里吃一碗汤圆，吃一口刚上市的香椿，或者是新鲜的茵陈。看灯和猜灯谜那就算了，我的生活里现在还缺老登吗？
祝你元宵节快乐，祝你也能找到元宵节的感觉。
