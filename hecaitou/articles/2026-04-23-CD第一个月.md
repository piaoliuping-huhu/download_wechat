# CD第一个月

- **日期**: 2026-04-23
- **链接**: https://www.hecaitou.com/2026/04/My-First-Month-with-CDs.html

---

有些事情你看别人做的时候会觉得很蠢，毫无任何道理。等到自己也去做一回，却发现穿越了经验和推理，去到了一个全新的世界。对于我而言，听 CD 就是其中一个例子。
今年 3 月 6 日那天，我从一位赤峰个人卖家处买了一台松下 SL-450 CD 随身听，电子古董，二手货，1997 年问世，距今接近 30 年。
完全没有道理，完全不符合逻辑---CD 是一种过时的音频载体，如今任何一家流媒体音乐平台都能提供更高码率的音乐文件。而且一张 CD 只能播放十几首歌曲，放完了还要换碟，人家流媒体只要有网有电就可以永不停歇地放下去。最后，你只能听 CD 上有的歌曲，做不到想听什么就可以立即播放。你可以在网上订购这首曲子对应的 CD 专辑，不过等你听到那是好几天之后的事情了。
所以，不管从什么角度看，在这个时代听 CD 都是一件非理性的事情。不过，人类刚巧就不是什么纯理性的动物，也包括我这一只。
我不单买了 CD 随身听，事实上我前后买了 5 台，真的是做到了每个房间最少有一台。我买了大概几十上百张 CD，没有具体数过，我只知道我已经买了三个透明塑料收纳盒，接下去可能要买 2 米 2 高的 CD 专用墙缝柜。
一切都发生得很自然，一丁点都不勉强，也没有情怀在硬撑。因为在过去的这一个月里，我每天都在听 CD，而且是一张张专辑从头听到尾。在这个过程里，我惊奇地发现，在不知不觉之中自己就从流媒体平台离开，一个月下来访问次数不超过 5 次。
之前我还担心，一旦离开了流媒体平台随机推荐歌曲功能，我就会彻底变成一个怀旧的老登，把自己青春期最喜欢听的专辑全部买一遍，然后眼含热泪反复听，回想当年和李翠花、王钢蛋她们的一幕又一幕。
没有这回事，我在网上随手下了很多单。一部分的确是我喜欢的旧日专辑，但也有很多我从未听过，只是看看封面，看看风格，再看看价格，感觉差不多就直接扔进购物车，最后再去找老板总体砍价。因此，我也听了不少「新」专辑。
音质是个问题吗？还好。从理论上来说，CD 的音质已经超越了大多数人的听力边界，也肯定超越了我的木耳。从实践上来说，我随手买了一堆 CD，正版均价 150，盗版、翻录 CD 均价 25，什么都没有的裸碟均价 5 元。横向比较下来，我认为我听不出什么差别。
价格、音质、内容都不是什么问题，我现在把CD 当做是一种新的计时器。每天早上起来，梳洗补妆完毕，喝下一杯黑咖啡，我就去选两张 CD 专辑，带着它们去到我的电脑桌。在我开始打字的同时，也开始播放第一张 CD。等到我检查完错别字，按下发送键，准备开始迎接留言到来时，第二张 CD 正好接近尾声。现在，我写一篇文章就可以明确说成是「两张 CD 的时间」。
进入了 CD 的世界，我才发现边听 CD 和写字这两件事之间其实很搭。在流媒体上，那些依据相同音乐基因做的歌单，只是个列表而已，不是作品。一张 CD 专辑就是一部完整的作品，其中有创作者的用意，让你先听什么，后听什么，然后构成你对作品的完整认知。写文章也是同样的，我不可能随意颠倒文章的段落次序，我也不可能把所有文章里的类似段落拼合成一整篇，文章刚好是现在这个样子，也只能是这个样子。
边听边写的感觉很好，就像是两个工匠分享同一件工作室，各自在工作台的一端搓自己的作品。我这边是我的键盘鼠标显示器，他那边是一台 CD 机、碟片以及音响设备。这样并行下来，我恍惚之间都有了一种自信的感觉：我这是在做「正经玩意儿」。
CD 的时节可能真的到来了。最近我看到国产厂商都在纷纷推出便携式 CD 机，闲鱼上的二手古董机除了几大旗舰之外，价格其实一直在缓慢下跌。CD 贩子们在网上又活跃了起来，我经常订货的一位甚至敲我小窗，试图把我发展为他的下线，在北京建个仓帮着他卖。
总体上听 CD 的人应该不会太多，但已经不是渐渐消亡的一个群体，目前有些抬头的迹象。经历过黑胶一张抢几百块的痛宰，经历过高清音乐文件网站的跑路，CD 对于我而言是个相对安心的选项。我还在听磁带，那是大刀长矛。我也听流媒体，那是飞机坦克。CD 是什么？CD 应该算是燧发枪吧？在这两者之间，一只脚跨入了当代的数字音乐。
那么，似乎也不能说我是怀旧的老登，只能说我是 21 世纪的火枪手。
